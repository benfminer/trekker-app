## Project Summary

TRACE Trekkers is a collective mileage-tracking web app for the TRACE school community. Students and staff log miles via an open (no-login) form; a small admin team manages submissions through a protected dashboard; cumulative progress displays on a live Mapbox map; and a site-based leaderboard pits Trace North, South, East, and West against each other. The app must launch April 1, 2026 — 12 days away — with ~7,135 miles of historical CSV data migrated and visible on day one. Phase 2 core features are largely built; remaining work is new features (leaderboard, admin self-service, password recovery, announcement email) plus QA and deploy.
