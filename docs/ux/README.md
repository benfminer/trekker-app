## Design System Summary

### Colors
| Token | Value | Use |
|-------|-------|-----|
| Background | #0a0a0a | All page backgrounds |
| Surface | #111111 | Cards, table rows |
| Border default | #1f1f1f | Dividers |
| Border strong | #2a2a2a | Input borders, card borders |
| Primary (orange) | #f97316 | CTAs, active states, route line, milestones |
| Text primary | #ffffff | Headings, key values |
| Text secondary | #9ca3af | Labels, supporting text |
| Text muted | #6b7280 | Placeholders, hints, timestamps |
| Error | #ef4444 | Validation, destructive actions |

### Typography
| Use | Font | Weights |
|-----|------|---------|
| Display / headings / CTAs | Oswald (Google Fonts) | 700, 500 |
| Body / labels / table data | Inter (Google Fonts) | 400, 500, 600 |

No monospace needed. No serif type. Oswald at 700 for all large display moments. Inter everywhere else.

### Breakpoints
| Name | Range |
|------|-------|

| Tablet | 640–1024px |
| Desktop | > 1024px |
